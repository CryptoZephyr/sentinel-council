HOW TO NORMALIZE SKILL OUTPUT
───────────────────────────────────────────────────────────────────────

Write a synchronous function:

    def normalize_skill_output(text: str) -> dict

Returns: {"score": int, "summary": str}

Logic in priority order:

  1. If text is empty or len(text) < 10:
     return {"score": 50, "summary": "No data"}

  2. Search for pattern: SIGNAL_SCORE: <number>
     If found:
       score = int(number), clamped 0-100
       summary = first sentence of text
       return {"score": score, "summary": summary}

  3. Count keyword occurrences:
     Bullish words: bullish, buy, uptrend, breakout, support, accumulation
     Bearish words: bearish, sell, downtrend, breakdown, resistance, distribution

     bull_count = matches in text.lower()
     bear_count = matches in text.lower()

     If bull + bear > 0:
         score = int(25 + (bull / (bull + bear)) * 50)
     Else:
         score = 50

     score = max(0, min(100, score))
     summary = text[:150]
     return {"score": score, "summary": summary}

RISK ENGINE
───────────────────────────────────────────────────────────────────────

Write a synchronous function:

    def calculate_position(
        decision: str,
        confidence: float,
        symbol: str,
        balance: float,
        open_positions: dict
    ) -> dict

  Returns dict with keys: action, size, reason

  HOLD decision          → action=HOLD, size=0, reason="Neutral signal"
  SELL + position exists → action=CLOSE, size=position["size"],
                           reason="Bearish — closing position"
  SELL + no position     → action=HOLD, size=0,
                           reason="Bearish but no position to close"
  BUY + already in pos   → action=HOLD, size=0,
                           reason="Already in position — no pyramiding"
  BUY + max positions    → action=HOLD, size=0,
                           reason="Max concurrent positions reached"
  BUY clear to open:
      risk_pct = RISK_AGGRESSIVE if confidence >= AGGRESSIVE_THRESHOLD
                 else RISK_CONSERVATIVE
      size = round(balance * risk_pct, 2)
      action=OPEN_LONG, size=size,
      reason=f"{risk_pct*100:.0f}% risk at {confidence:.1f}% confidence"

───────────────────────────────────────────────────────────────────────
SIMULATED PORTFOLIO
───────────────────────────────────────────────────────────────────────

Write a class SimPortfolio:

  __init__:
    balance = Config.STARTING_BALANCE
    open_positions = {}
    trades = []
    total_pnl = 0.0
    win_count = 0
    trade_count = 0

  get_price(symbol) -> float:
    GET https://api.bitget.com/api/v2/mix/market/ticker
        ?symbol={symbol}&productType=USDT-FUTURES
    Parse response["data"][0]["lastPr"] as float
    Return 0.0 on any failure

  open_long(symbol, size) -> dict:
    price = self.get_price(symbol)
    if price == 0.0: log warning, return {}
    self.balance -= size
    store position with size, entry_price, opened_at
    log and return position

  close_position(symbol) -> dict:
    pop position
    price = get_price (fallback to entry_price if 0)
    pnl = (price - entry_price) / entry_price * size
    balance += size + pnl
    total_pnl += pnl
    update win_count and trade_count
    log and return trade record

  execute(risk_result, symbol) -> dict:
    dispatch to open_long or close_position based on action

  summary() -> dict:
    return balance, open_positions, total_pnl,
           trade_count, win_count, win_rate
